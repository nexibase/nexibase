# PG Promo (sir.kr) Widget — Design

## Purpose

Add a promotional dashboard widget that advertises [sir.kr/services](https://sir.kr/services) — a Korean PG (payment gateway) reseller — to Nexibase operators. Nexibase is a CMS for communities and shops; anyone spinning up a shop needs a PG, so this widget targets a real need while monetizing the platform through affiliate-style promotion.

## Source of claims

`sir.kr/services` (confirmed by fetch on 2026-04-24) offers:

- KG이니시스 credit card fee **2.80%** (lowest domestically)
- Lifetime annual fee waiver
- Recurring billing (정기결제) and instant-open (바로오픈) support
- Bundled identity verification (본인인증) and SMS

The widget's numeric claim (2.80%) and benefits list derive from this page.

## Scope

### In

- One React widget component, dashboard-embeddable like other `src/widgets/*`
- Korean and English copy via `next-intl`
- External link to `https://sir.kr/services` with `rel="sponsored noopener"`, opens new tab
- Auto-registered by `scripts/scan-plugins.js` (no manual registry edit)

### Out (YAGNI)

- Per-user dismiss / localStorage hide — admin can remove the widget instead
- Admin-editable settings (`settingsSchema: null`) — copy is hardcoded; if the reseller's numbers change, update the code
- PG comparison table — this is a promo, not a comparison site
- Unit tests — project has no widget-level test infra; visual QA via `npm run dev` instead

## Component (`PgPromoSir.tsx`)

- `"use client"` directive (consistent with sibling widgets)
- Uses `Card` / `CardContent` from `@/components/ui/card`
- No data fetching, no loading state, no effects — pure presentational
- `useTranslations('widgets')` for text; numeric "2.80%" is hardcoded (not translated)
- Icons from `lucide-react`: `CreditCard` (header), `Check` (benefit bullets), `ArrowRight` (CTA)
- CTA is a plain `<a href="https://sir.kr/services" target="_blank" rel="sponsored noopener">` — not `next/link`, because it's an external sponsored link

### Visual layout (top to bottom)

```
[CreditCard icon]  headline (small, muted)
                   leadLabel
                   2.80%                     ← hero number
[Check] bullet1
[Check] bullet2
[Check] bullet3
[CTA button → sir.kr/services]
```

### Styling notes

- Background: `bg-gradient-to-br from-primary/10 via-primary/5 to-background border-primary/20` — matches `WelcomeBanner` for dashboard tonal consistency
- Hero number: `text-4xl md:text-5xl font-extrabold`
- Bullets: `text-sm` with `Check h-4 w-4` icons
- Outer layout: `h-full flex flex-col justify-between p-6`

## Metadata (`PgPromoSir.meta.ts`)

```ts
export default {
  title: 'PG Promo (sir.kr)',
  description: 'Promotional widget for sir.kr/services low payment gateway fees',
  defaultZone: 'side',
  defaultColSpan: 1,
  defaultRowSpan: 2,
  settingsSchema: null,
}
```

`defaultZone: 'side'` because promos belong in the sidebar, not the hero slot. `1×2` gives a tall narrow card — the vertical space holds the hero number + 3 bullets + CTA without cramping.

| `bullet1` | 연회비 평생면제 | Lifetime annual-fee waiver |
| `bullet2` | 정기결제 · 바로오픈 지원 | Recurring billing · instant open |

The English copy will be seen by non-Korean Nexibase users in the open-source community; it honestly frames the service as a Korean-market offering without overselling.

## Error handling

None required:

- No network calls
- No user input
- Missing translation keys fall through to next-intl default behavior

## Testing

Manual verification only:

1. `npm run dev`
2. Admin → add `PgPromoSir` widget to a layout
3. Confirm render in both `ko` and `en` locales
4. Click CTA — opens `sir.kr/services` in new tab
5. Inspect link attributes — confirm `rel="sponsored noopener"` present
